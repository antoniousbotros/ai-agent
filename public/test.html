Welcome to your Rabeh Test Site

If your AI backend and widget are working correctly, you should see the floating chat button in the bottom right!

How to make it work:

  1. Ensure Ollama is running on your Mac (ollama run gemma:2b)
  2. In Supabase, create a Bot row in the bots table, give it a System Prompt, and copy its UUID.
  3. In Supabase, create an API key row in the api_keys table, and copy the KEY_STRING.
  4. Edit `public/test.html` and replace the YOUR_API_KEY and YOUR_BOT_ID below!
